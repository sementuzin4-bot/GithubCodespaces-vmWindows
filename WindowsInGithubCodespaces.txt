1.Create github codespace (https://github/codespaces)

2.Create a folder "win"

3.Create a "win.yml" file and enter the command
   Command:

services:
  windows:
    image: dockurr/windows
    container_name: windows
    environment:
      VERSION: "Custom disk (write the address of the iso file) or the version below" (be sure to enter!)
      USERNAME: "random_user" (be sure to enter!)
      PASSWORD: "random_password" (be sure to enter!)
      RAM_SIZE: "10G"
      CPU_CORES: "4"
      DISK_SIZE: "32G"
    devices:
      - /dev/kvm
      - /dev/net/tun
    cap_add:
      - NET_ADMIN
    ports:
      - "8006:8006"
      - "3389:3389/tcp"
      - "3389:3389/udp"
    stop_grace_period: 2m

4.Move the "win.yml" file to the "win" folder

5.Enter the command:
cd win 
sudo su
sudo apt-get update
sudo apt-get upgrade -y
sudo apt install docker.io docker-compose
docker
pwd

6.Start RDP:
sudo docker-compose -f win.yml up
  Restart RDP:
sudo docker-compose -f win.yml restart 
  Shutdown RDP:
sudo docker-compose -f windows10.yml shutdown
